Notas de Atualização - Versão 4.0

Mudanças no código como um todo

Foi mudado boa parte do backend do código para que funções como jogos e foto de perfil fossem possiveís de serem aplicadas.

Página inicial

Uma página principal foi adicionada para que os usuários pudessem entrar em outras abas, como suporte, patch-notes e o login.

Download

Foi adicionada uma área de download, onde disponibilizamos o código aberto de forma fácil para qualquer um que queira nos ajudar.



Notas de Atualização - Versão 3.0

Mudanças na Estética do Site

Foi implementada uma nova estética e um novo nome para o site, focando em novos objetivos e novos públicos, entregando um produto diferente para os usuários.

Novas Funções

Alteramos o chat, trazendo novas funcionalidades que desejávamos: agora temos emojis e suporte para envio de todos os tipos de arquivos.

Jogos

Iniciamos o desenvolvimento e implementação de jogos no nosso site, oferecendo uma opção diferenciada para os usuários.

Divisão Interna do Código

Reestruturamos o código-fonte da maioria das páginas, para um melhor carregamento, evitando problemas como objetos se sobrepondo e CSS instável.



Notas de Atualização - Versão 2.0

Melhorias no Chat

Implementamos diversas funções na aba de chat, melhorando a conexão entre usuários e oferecendo uma experiência mais fluida.

Melhorias na Interface de Usuário

Alteramos partes da interface de login e registro, dando mais liberdade às necessidades dos usuários.

Início de Novas Funcionalidades

Iniciamos a implementação de funções para envio de imagens e outros documentos, facilitando envios rápidos.



Notas de Atualização - Versão 1.0

Melhorias na Interface

Com a versão 1.0, introduzimos uma série de mudanças na interface para tornar a navegação mais fluida. Atualizamos a paleta de cores e ajustamos ícones para maior clareza e usabilidade.

Correção de Bugs

Corrigimos falhas críticas como erros intermitentes no login, desconexões inesperadas e problemas com a alternância de temas, proporcionando uma experiência mais estável.

Novos Recursos

Adicionamos novas telas e opções de personalização no chat, melhorando a experiência geral do usuário.

Melhorias Internas

Otimizamos o código interno, melhorando o carregamento do site e resolvendo problemas de travamentos.
